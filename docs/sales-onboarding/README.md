# Polaris Point — Sales Onboarding

Welcome. This folder is everything a new salesperson needs to start booking and closing Polaris Point clients.

Read in this order:

1. **[onboarding-deck.html](./onboarding-deck.html)** — open in a browser. Branded slide deck covering who we are, what we sell, who we sell to, and how we sell. Start here on day one.
2. **[services-and-pricing.md](./services-and-pricing.md)** — every product we offer, what's included at each tier, and the prices. Reference doc; keep it open during calls.
3. **[sales-playbook.md](./sales-playbook.md)** — the full flow from cold prospect to closed deal: outreach scripts, discovery questions, demo flow, proposal templates, and objection handling.
4. **[commission-term-sheet.md](./commission-term-sheet.md)** — your compensation, how sales are counted, when you get paid, and what triggers a clawback. Sign before your first deal.
